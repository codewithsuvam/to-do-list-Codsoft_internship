# 📝 To-Do List Application

A modern To-Do List application built using Python and Tkinter. This project helps users manage daily tasks efficiently with features like task creation, editing, completion tracking, and data persistence.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Mark all tasks as completed
- Filter tasks (All, Pending, Completed)
- Save tasks automatically using JSON
- Modern and responsive UI

## 🛠 Technologies Used

- Python
- Tkinter
- JSON

├── todo_app.py
├── tasks.json
└── README.md
